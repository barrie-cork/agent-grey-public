Understanding Search Statistics
  • Provider: The search API used for each execution (e.g., Serper.dev for Google, SearchAPI.io for Bing)
  • Completion Rate: Percentage of queries that have at least one successful execution
  • Overlap: Results found by multiple providers, indicating cross-provider redundancy
  • Deduplication: Duplicate results removed during processing, shown in the Total Results card
  • Success Rate: Percentage of successful executions vs. total execution attempts (includes retries)
  • Rate Limiting: Local rate limiter set to 50 requests/minute to prevent API quota exhaustion while allowing full pagination
