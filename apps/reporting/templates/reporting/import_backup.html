How Import Works
  1. Select the Excel file you edited offline (must be a .xlsx file from this session)
  2. Click "Import Changes" to upload the file
  3. The system will validate your changes and check for errors
  4. Valid changes will be synced to the database automatically
  5. You'll see a summary of updated records and any errors
What Gets Imported

The import process will sync the following changes from your Excel file:

  • Review Decision: Include, Exclude, or Maybe
  • Exclusion Reason: For excluded results only
  • Notes: Any notes you've added during review

ℹ️ The system matches Excel rows to database records by title and URL. If a result can't be matched, it will be skipped.
